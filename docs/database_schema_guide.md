# 数据库字段说明文档

数据来源：titan007.com 足球数据平台
更新时间：2026-03-08

---

## 整体概览

本数据库共存储 **7 类数据**，彼此关联如下：

- **联赛信息** 是赛事的背景分类，一个联赛对应多场比赛
- **球队信息** 记录参赛队伍，每场比赛有一支主队和一支客队
- **赛事列表** 是核心，记录每场比赛的基本信息和比赛结果
- **联赛排名详情** 是每场比赛前两支球队在联赛中的积分榜表现
- **博彩公司** 是提供赔率数据的机构字典
- **欧赔快照** 记录各家博彩公司对这场比赛给出的最新赔率
- **赔率历史** 记录某家公司对某场比赛的赔率从开盘到临场的每一次变动

## 四、联赛排名详情

记录比赛双方在联赛积分榜上的详细数据。每场比赛会产生 **16 条记录**，覆盖以下所有维度的组合：

**队伍**：主队 / 客队

**统计时段**：全场 / 半场

**统计范围**：
- 全部场次（赛季至今所有比赛）
- 主场场次（仅统计在主场踢的比赛）
- 客场场次（仅统计在客场踢的比赛）
- 近 6 场（最近 6 场比赛）

### 字段说明

| 字段名 | 含义 | 示例 |
|--------|------|------|
| 比赛编号 | 关联到哪场比赛 | `2915935` |
| 队伍角色 | `home`=主队，`away`=客队 | `home` |
| 统计时段 | `ft`=全场，`ht`=半场 | `ft` |
| 统计范围 | `total`=全部，`home`=主场，`away`=客场，`last6`=近6场 | `total` |
| 已赛场次 | 该统计范围内已经踢了多少场 | `22` |
| 胜场数 | 赢了多少场 | `14` |
| 平场数 | 平了多少场 | `4` |
| 负场数 | 输了多少场 | `4` |
| 进球数 | 共进了多少球 | `38` |
| 失球数 | 共失了多少球 | `18` |
| 净胜球 | 进球数 - 失球数 | `20` |
| 积分 | 当前积分榜积分 | `46` |
| 排名 | 当前联赛排名 | `6` |
| 胜率 | 胜场占已赛场次的比例 | `0.636`（即 63.6%） |
| 数据抓取时间 | 本条记录获取时间 | `2026-03-08 12:55:00` |

## 六、欧赔快照

记录每场比赛中各家博彩公司给出的赔率，每场比赛每家公司一条记录。

> **什么是欧赔？** 欧式赔率（European Odds）直接表示赔倍数，例如主胜赔率 2.40 表示押注 1 元赢得 2.40 元（含本金）。返还率是三个赔率倒数之和，越接近 100% 表示公司抽水越少。

> **什么是凯利指数？** 凯利指数（Kelly Index）是衡量赔率高低的参考指标。高于 1 表示赔率偏高（对应结果可能性被高估），低于 1 表示赔率偏低。通常用来辅助判断赔率的变动方向和市场倾向。

### 基本信息

| 字段名 | 含义 | 示例 |
|--------|------|------|
| 记录编号 | 这条赔率记录的唯一编号（用于查询历史变动） | `151025873` |
| 比赛编号 | 关联到哪场比赛 | `2921107` |
| 博彩公司 | 关联到哪家博彩公司 | William Hill |
| 最近变动时间 | 该公司最近一次调整赔率的时间 | `2026-03-06 23:30:00` |
| 数据抓取时间 | 本条记录获取时间 | `2026-03-08 12:55:00` |

### 初盘赔率

> 初盘是博彩公司在比赛开盘时首次发布的赔率，反映公司最初的判断，后续可能会随市场资金流向调整。

| 字段名 | 含义 | 示例 |
|--------|------|------|
| 初盘主胜赔率 | 开盘时主队获胜的赔率 | `2.63` |
| 初盘平局赔率 | 开盘时平局的赔率 | `2.90` |
| 初盘客胜赔率 | 开盘时客队获胜的赔率 | `2.50` |
| 初盘主胜概率 | 开盘时主胜的隐含概率 | `33.80%` |
| 初盘平局概率 | 开盘时平局的隐含概率 | `30.65%` |
| 初盘客胜概率 | 开盘时客胜的隐含概率 | `35.55%` |
| 初盘返还率 | 开盘时三项赔率的整体返还率 | `88.88%` |

### 即时赔率

> 即时赔率是临近比赛时公司给出的最新赔率，通常经过多次调整。

## 七、赔率历史变动

记录某场比赛中某家博彩公司从开盘到临场每一次赔率调整的快照，按时间从新到旧排列。

> 通过对比初盘和终盘赔率的变化，可以判断市场资金的流向（赔率下降的方向通常代表大量资金押注）。

| 字段名 | 含义 | 示例 |
|--------|------|------|
| 所属赔率记录 | 关联到"欧赔快照"中的哪条记录 | `151025873` |
| 比赛编号 | 关联到哪场比赛 | `2921107` |
| 博彩公司编号 | 关联到哪家博彩公司 | `115`（William Hill） |
| 主胜赔率 | 本次调整后的主胜赔率 | `2.40` |
| 平局赔率 | 本次调整后的平局赔率 | `3.00` |
| 客胜赔率 | 本次调整后的客胜赔率 | `2.70` |
| 主胜概率 | 本次调整后主胜的隐含概率 | `37.19%` |
| 平局概率 | 本次调整后平局的隐含概率 | `29.75%` |
| 客胜概率 | 本次调整后客胜的隐含概率 | `33.06%` |
| 返还率 | 本次调整后的整体返还率 | `89.26%` |
| 主胜凯利指数 | 本次调整后主胜的凯利值 | `0.85` |
| 平局凯利指数 | 本次调整后平局的凯利值 | `0.89` |
| 客胜凯利指数 | 本次调整后客胜的凯利值 | `0.94` |
| 变动时间 | 本次赔率调整发生的时间 | `2026-03-07 07:18` |
| 是否初盘 | 这条记录是否为该公司最早发布的开盘赔率 | 是 / 否 |
| 主胜变动方向 | 与上一次相比主胜赔率是升还是降 | 升高 / 降低 / 不变 |
| 平局变动方向 | 与上一次相比平局赔率的变动方向 | 升高 / 降低 / 不变 |
| 客胜变动方向 | 与上一次相比客胜赔率的变动方向 | 升高 / 降低 / 不变 |
