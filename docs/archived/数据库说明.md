# Quant 数据库内容说明

> 本文档面向非技术用户，用通俗语言介绍数据库中存储了哪些内容、每张表的用途以及各字段含义。
a
---

## 数据来源

所有数据均抓取自 **titan007.com** 足球数据平台，包括比赛信息、联赛积分、欧洲赔率、亚盘赔率等。

---

## 数据库共有 10 张表

| 表名 | 中文名称 | 简介 |
|------|----------|------|
| `leagues` | 联赛字典 | 存储联赛的名称、颜色等基本信息 |
| `teams` | 球队字典 | 存储球队的名称（中文、英文） |
| `matches` | 比赛列表 | 每场比赛的基本信息，如时间、比分、红黄牌等 |
| `match_standings` | 赛前积分榜 | 比赛前两队在联赛中的排名和成绩统计 |
| `match_recent` | 近期战绩 | 比赛前两队最近几场比赛的历史记录 |
| `companies` | 博彩公司字典 | 博彩公司名称列表（如 Bet365、William Hill） |
| `match_odds` | 欧洲赔率快照 | 各博彩公司对这场比赛开出的欧赔（初盘 & 即时） |
| `odds_history` | 欧赔变动历史 | 欧洲赔率的每一次变动记录 |
| `match_asian_odds` | 亚盘赔率快照 | 各博彩公司对这场比赛开出的亚盘（初盘 & 即时） |
| `asian_odds_history` | 亚盘变动历史 | 亚盘赔率的每一次变动记录 |

## 各表详细说明

### 1. 联赛字典（leagues）

存储所有联赛的基础信息，每条联赛只存一次，不会重复。

| 字段 | 说明 | 示例 |
|------|------|------|
| league_abbr | 联赛英文缩写（唯一标识） | `JPN D1`、`ENG PL` |
| league_name_cn | 联赛中文名称 | `日职联`、`英超` |
| league_color | 在界面上显示的颜色代码 | `#009900` |
| country_id | 所属国家编号 | `81` |

---

### 2. 球队字典（teams）

存储所有球队的基础信息，每支球队只存一次。

| 字段 | 说明 | 示例 |
|------|------|------|
| team_id | 球队唯一编号（来自 titan007） | `12345` |
| team_name_cn | 球队中文名 | `切尔西` |
| team_name_en | 球队英文名 | `Chelsea` |

---

### 3. 比赛列表（matches）

每场比赛对应一条记录，记录比赛的完整信息。

| 字段 | 说明 | 示例 |
|------|------|------|
| schedule_id | 比赛唯一编号 | `3001234` |
| match_time | 比赛开始时间 | `2026-03-26 20:00` |
| status | 比赛状态 | 见下表 |
| league_abbr | 所属联赛 | `ENG PL` |
| home_team_id | 主队编号 | `101` |
| home_rank | 主队赛前联赛排名 | `3` |
| away_team_id | 客队编号 | `202` |
| away_rank | 客队赛前联赛排名 | `7` |
| home_score | 主队全场进球 | `2` |
| away_score | 客队全场进球 | `1` |
| home_half_score | 主队半场进球 | `1` |
| away_half_score | 客队半场进球 | `0` |
| home_red_cards | 主队红牌数 | `0` |
| away_red_cards | 客队红牌数 | `1` |
| home_yellow_cards | 主队黄牌数 | `2` |
| away_yellow_cards | 客队黄牌数 | `3` |
| fetched_at | 数据抓取时间 | `2026-03-26 21:05` |

**比赛状态说明：**

| 状态值 | 含义 |
|--------|------|
| `0` | 未开赛 |
| `1` | 上半场进行中 |
| `3` | 下半场进行中 |
| `-1` | 已完场 |
| `-11` | 比赛中断 |
| `-12` | 比赛腰斩（未打完） |
| `-14` | 比赛推迟 |

---

### 4. 赛前积分榜（match_standings）

记录比赛前两队在联赛中的成绩统计，每场比赛最多生成 **16 条**统计数据（主/客队 × 全场/半场 × 总/主场/客场/近6场）。

| 字段 | 说明 | 示例 |
|------|------|------|
| schedule_id | 对应的比赛编号 | `3001234` |
| side | 主队还是客队 | `home` / `away` |
| period | 统计的是全场还是半场 | `ft`（全场）/ `ht`（半场） |
| scope | 统计范围 | `total`（全部）/ `home`（主场）/ `away`（客场）/ `last6`（近6场） |
| played | 已赛场次 | `20` |
| win | 胜场数 | `10` |
| draw | 平场数 | `5` |
| loss | 负场数 | `5` |
| goals_for | 进球数 | `32` |
| goals_against | 失球数 | `18` |
| goal_diff | 净胜球 | `14` |
| points | 积分 | `35` |
| rank | 联赛排名 | `3` |
| win_rate | 胜率（小数） | `0.50`（表示 50%） |

---

### 5. 近期战绩（match_recent）

记录比赛前，两队各自最近几场比赛的历史对阵记录，用于参考近期状态。

| 字段 | 说明 | 示例 |
|------|------|------|
| schedule_id | 当前比赛编号 | `3001234` |
| side | 当前比赛中是主队还是客队 | `home` / `away` |
| match_id | 历史比赛的编号 | `2998765` |
| date | 历史比赛日期 | `2026-03-10` |
| league | 历史比赛所在联赛 | `ENG PL` |
| home_name | 历史比赛主队名称 | `切尔西` |
| away_name | 历史比赛客队名称 | `阿森纳` |
| home_ft | 历史比赛主队全场得分 | `2` |
| away_ft | 历史比赛客队全场得分 | `1` |
| ht_score | 历史比赛半场比分 | `1-0` |
| handicap | 该场比赛亚让盘线 | `-0.5` |
| result | 胜负结果（相对当前队伍） | `1`（赢）/ `0`（平）/ `-1`（输） |
| hc_result | 亚盘胜负结果 | `1`（赢盘）/ `-1`（输盘） |

---

### 6. 博彩公司字典（companies）

存储所有博彩公司的名称，每家公司只存一次。

| 字段 | 说明 | 示例 |
|------|------|------|
| company_id | 公司唯一编号 | `36` |
| company_name | 公司名称 | `Bet 365` |
| label | 分类标签 | `36*(英国)` |

---

### 7. 欧洲赔率快照（match_odds）

记录每家博彩公司对某场比赛开出的欧洲赔率（主胜、平局、客胜），包含**初盘**（开盘时）和**即时**（最新）两组数据。

| 字段 | 说明 | 示例 |
|------|------|------|
| record_id | 赔率记录唯一编号 | `98765` |
| schedule_id | 对应的比赛编号 | `3001234` |
| company_id | 对应的博彩公司编号 | `36` |
| open_win | 初盘主胜赔率 | `2.10` |
| open_draw | 初盘平局赔率 | `3.40` |
| open_lose | 初盘客胜赔率 | `3.60` |
| open_win_prob | 初盘主胜隐含概率 | `47.6%` |
| open_draw_prob | 初盘平局隐含概率 | `29.4%` |
| open_lose_prob | 初盘客胜隐含概率 | `27.8%` |
| open_payout_rate | 初盘返还率 | `94.8%` |
| cur_win | 即时主胜赔率 | `1.95` |
| cur_draw | 即时平局赔率 | `3.50` |
| cur_lose | 即时客胜赔率 | `3.90` |
| cur_win_prob | 即时主胜隐含概率 | `51.3%` |
| cur_draw_prob | 即时平局隐含概率 | `28.6%` |
| cur_lose_prob | 即时客胜隐含概率 | `25.6%` |
| cur_payout_rate | 即时返还率 | `95.5%` |
| kelly_win | 凯利指数（主胜） | `1.05` |
| kelly_draw | 凯利指数（平局） | `0.97` |
| kelly_lose | 凯利指数（客胜） | `0.88` |
| change_time | 最近一次赔率变动时间 | `2026-03-26 18:30:00` |

> **什么是凯利指数？** 凯利指数 > 1 说明该结果赔率相对赛果概率偏高（庄家看好），< 1 说明偏低。

---

### 8. 欧赔变动历史（odds_history）

记录欧洲赔率的每一次变动，可以追溯某家公司对某场比赛赔率的完整变化轨迹。

| 字段 | 说明 | 示例 |
|------|------|------|
| record_id | 关联到欧赔快照的编号 | `98765` |
| schedule_id | 比赛编号 | `3001234` |
| company_id | 博彩公司编号 | `36` |
| change_time | 本次赔率变动的时间 | `2026-03-25 10:00` |
| win / draw / lose | 本次赔率值 | `2.05 / 3.40 / 3.70` |
| win_prob / draw_prob / lose_prob | 本次隐含概率 | `48.8% / 29.4% / 27.0%` |
| payout_rate | 本次返还率 | `95.2%` |
| kelly_win / kelly_draw / kelly_lose | 本次凯利指数 | `1.02 / 0.98 / 0.91` |
| is_opening | 是否为初盘 | `1`（是）/ `0`（否） |
| win_dir | 主胜赔率变动方向 | `up`（升）/ `down`（降）/ `unchanged`（未变） |
| draw_dir | 平局赔率变动方向 | 同上 |
| lose_dir | 客胜赔率变动方向 | 同上 |

---

### 9. 亚盘赔率快照（match_asian_odds）

记录每家博彩公司对某场比赛开出的亚洲让球盘赔率，包含**初盘**和**即时**两组数据。

| 字段 | 说明 | 示例 |
|------|------|------|
| schedule_id | 对应的比赛编号 | `3001234` |
| company_id | 对应的博彩公司编号 | `36` |
| open_handicap | 初盘让球线 | `-0.5`（主让半球）/ `+0.25` |
| open_home | 初盘主队水位（赔率） | `0.92` |
| open_away | 初盘客队水位（赔率） | `0.98` |
| cur_handicap | 即时让球线 | `-0.75` |
| cur_home | 即时主队水位 | `0.88` |
| cur_away | 即时客队水位 | `1.00` |

> **什么是水位？** 水位越低，庄家认为该队赢的概率越小（要扣更多手续费）；水位越接近 1，赔率越公允。

---

### 10. 亚盘变动历史（asian_odds_history）

记录亚盘赔率的每一次变动，可追溯让球线和水位的完整变化轨迹。

| 字段 | 说明 | 示例 |
|------|------|------|
| schedule_id | 比赛编号 | `3001234` |
| company_id | 博彩公司编号 | `36` |
| change_time | 本次变动时间 | `2026-03-25 14:00` |
| score | 变动时的当前比分 | `1-0` |
| handicap | 本次让球线 | `-0.5` |
| home_odds | 本次主队水位 | `0.90` |
| away_odds | 本次客队水位 | `0.98` |
| is_opening | 是否为初盘 | `1`（是）/ `0`（否） |
| home_dir | 主队水位变动方向 | `down`（降）/ `up`（升）/ `unchanged`（未变） |
| away_dir | 客队水位变动方向 | 同上 |

## 附：表与表之间的关系

```
leagues ←──── matches ────→ teams
                  │
          ┌───────┼───────┐
          ↓       ↓       ↓
   match_standings  match_recent  match_odds ─→ odds_history
                              match_asian_odds ─→ asian_odds_history

companies ─────────────────────────────────→ match_odds
                                          → match_asian_odds
```

- 一场 **比赛** 属于一个 **联赛**，有一支 **主队** 和一支 **客队**
- 每场比赛有多条 **积分榜**数据（多个统计维度）
- 每场比赛有多条 **近期战绩**（两队各自的历史记录）
- 每场比赛有多家 **公司**的赔率，每家公司的赔率有完整的**变动历史**
